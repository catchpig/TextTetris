# TextTetris - 文字俄罗斯方块游戏设计

## 1. 项目概述

**项目名称**：TextTetris（文字俄罗斯方块）

**核心功能**：一款以大字符艺术风格呈现的俄罗斯方块游戏，玩家通过手势操控方块下落、旋转和移动，目标是在保证棋盘不填满的情况下尽可能消除更多行获得高分。

**目标用户**：Android 手机用户，喜欢休闲益智类游戏的玩家

---

## 2. 技术架构

采用 **MVVM + 领域驱动设计（DDD）**，分为三层：

```
┌─────────────────────────────────────┐
│           UI Layer (ui)             │
│  Compose Screen + ViewModel         │
├─────────────────────────────────────┤
│         Domain Layer (domain)        │
│  Use Cases + Entities + Repository  │
├─────────────────────────────────────┤
│         Data Layer (data)            │
│  Repository Impl + Room Database    │
└─────────────────────────────────────┘
```

**包结构**：
- `com.texttetris.ui.screens` — 游戏界面
- `com.texttetris.ui.components` — 可复用 UI 组件
- `com.texttetris.ui.viewmodel` — ViewModel
- `com.texttetris.domain.model` — 领域模型（方块、棋盘、游戏状态）
- `com.texttetris.domain.usecase` — 用例（移动、旋转、消行、计分）
- `com.texttetris.domain.repository` — 仓库接口
- `com.texttetris.data.repository` — 仓库实现
- `com.texttetris.data.local` — Room 数据库

---

## 3. 核心领域模型

### GameBoard（棋盘）
- 10 列 × 20 行
- 用 `List<List<Cell?>>` 表示，`null` 表示空，有颜色表示已填充

### Tetromino（方块）
- 7 种经典形状：I、O、T、S、Z、J、L
- 每个方块有颜色、形状矩阵、当前位置

### GameState（游戏状态）
- `idle` — 等待开始
- `playing` — 游戏中
- `paused` — 暂停
- `gameOver` — 游戏结束

### Score（分数）
- 每次消行获得分数
- 1 行 = 100 分，2 行 = 300 分，3 行 = 500 分，4 行 = 800 分

## 4. UI 设计

### 布局结构
```
┌────────────────────────────────┐
│      TextTetris (Title)        │
├────────────────────────────────┤
│  Score: 1200    Level: 3      │
├────────────────────────────────┤
│                                │
│      ┌────────────────┐         │
│      │                │  NEXT  │
│      │   GAME AREA    │  ┌───┐ │
│      │   (Canvas)     │  │ T │ │
│      │                │  └───┘ │
│      │                │         │
│      └────────────────┘         │
│                                │
├────────────────────────────────┤
│   滑动操作提示区域              │
└────────────────────────────────┘
```

### Canvas 绘制
- 每个单元格用大号空心方块字符 `□` 绘制
- 已填充的方块用带颜色填充的字符表示
- 活力彩色配色：I型=青色、O型=黄色、T型=紫色、S型=绿色、Z型=红色、J型=蓝色、L型=橙色

### 手势识别
- 左滑：向左移动
- 右滑：向右移动
- 上滑：旋转
- 下滑：快速下落
- 双击：暂停/继续

## 5. 音效设计

| 事件 | 音效类型 |
|------|----------|
| 方块移动 | 短促点击音 |
| 方块旋转 | 轻微嗡嗡声 |
| 消行 | 清爽消除音 |
| 游戏结束 | 下降音调 |

使用 Android `SoundPool` 播放简短音效文件。

## 6. 数据存储

使用 **Room** 数据库存储游戏记录：

### 数据库设计

```
GameRecord (游戏记录表)
├── id: Long (主键，自增)
├── score: Int (本次得分)
├── level: Int (达到的等级)
├── linesCleared: Int (消行数)
├── duration: Long (游戏时长，毫秒)
├── playedAt: Long (时间戳)
```

### Repository 接口
```kotlin
interface GameRepository {
    suspend fun saveGameRecord(record: GameRecord)
    suspend fun getAllRecords(): List<GameRecord>
    suspend fun getHighScore(): Int
    suspend fun getTotalGames(): Int
    suspend fun getTotalLinesCleared(): Int
}
```

### DAOs
- `GameRecordDao` — 插入记录、查询所有、按分数排序
